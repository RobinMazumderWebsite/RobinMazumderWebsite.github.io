Fire Escape

Winter 2015

For my second HackED hackathon, I tried to go bigger than ever. Knowing that hardware hacks that are rooted in reality usually do win hackathons, I assembled a team that would be able to design and build an arcade game with an element of gamifying education. As a team we came up with the idea that a puzzle game where the user places fire safety items where the AI people can find them to save themselves when the fire starts until the fire truck can get there to save them.

The game was built with SDL C++, which is much more low level than I'm used to. I was tasked with the UI of the game and had a lot of work trying to create something that looked nice with a minimum amount of tools.

Fire Escape was made to be an arcade game and an arcade box and hardware controls were built for it for the hackathon. However the software was buggy and our presentation did not appeal to the judges so we didn't place.

ReviewApp Template

Summer 2015

The idea behind this project comes from the realization I had that people's opinions are the best form of qualitative data for many things in our world. Applications like Rotten Tomatoes and Yelp have become very popular offering this simple service to a topic of need (movies and restaurants respectively). This project is my attempt to create such an application for a now unknown/undecided topic of need.

The template uses ASP.NET's MVC framework and Entity Framework. I used .NET's scaffolding to quickly create the front end and used Bootstrap for some looks. Entity Framework took care of all the object-relational mapping - what data goes where and is related to other data how. I based this project on a tutorial from ASP.NET's website, it can be found here.

Conference app

Summer 2014

I built a mobile Confrenece app while I worked for fourthdimension inc. as an intern. The mobile webapp was created using the Ionic Framework and Apache Cordova. The app was built to provide attendees with the conference schedule, details on the seminars and presenters, maps of the area and provided internal staff with user feedback on seminars and to be a template for future years.

The application handled all this data with read and write capabilities to an Azure Mobile Service and some JavaScript services. I worked with coworkers to create the user interface of the application, which was built with SCSS and CSS.

The project required me to learn and deal with Angular.js framework, Ionic framework, Command Line, GIT, CSS some JavaScript and Azure Databases. It taught me lessons about working with a team in an office environment. Unfortunately, since the application was created under contract, I do not have access to point you to any of the work I did.
